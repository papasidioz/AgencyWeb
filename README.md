## 💻 Sobre o projeto

<!-- EXPLICA O MOTIVO DO PROJETO -->
AgencyWeb foi um projeto desenvolvido durante o curso de Desenvolvedor Front-End do SENAI com o objetivo de praticar e aprofundar meus conhecimentos em HTML e CSS. Através da criação de uma landing page para uma agência digital, pude aplicar conceitos como estruturação de páginas, estilização de elementos, organização de layouts e responsividade. Esse projeto foi importante para fortalecer minha base no desenvolvimento web e acompanhar minha evolução na construção de interfaces modernas e funcionais.
